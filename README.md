## Prerequisite

FFmpeg is required and must be installed.

| OS | How to install |
|-----------|-------------|
| Ubuntu/Debian | `sudo apt install ffmpeg` |
| MacOS | `brew install ffmpeg` |
| Windows | Download .exe from https://ffmpeg.org/download.html and add to PATH |
